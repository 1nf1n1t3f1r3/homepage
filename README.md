# Readme

This project has been quite interesting. it's one of the first times I'm building something that I'd like to put (and keep) online. That has made it a lot more motivating to work on.

## Tech Stack

There's not a _whole_ lot to say about what went into making this. There's no backend, so it's very much a vanilla React app. React's taken me a long while to start to figure out, but it's relatively smooth to work with.

## Design

Probably the most difficult part about getting started is choosing what the design is going to be like. I didn't have a very strong opinion about it. I always browse on Dark Mode, so of course it was going to be dark. Starting by simply making a Header helps a lot. I also wanted the tabs to have a bit of a different feel. At first I wanted to host everything under janusdevries.nl, before I changed it to sub-domains, or even separate domains, so a different style would help set things apart. Then I remembered I still needed an introductory text to go with the cards, so I made an extra box. 'Nuff said.

### Odin Tab

First, I tried to mimic the Odin style, using their colours with a colour picker and their fonts. Unfortunately, it wound up looking very uncanny; it's really easy to tell that it's just slightly 'off'. So, I threw that out after a lot of fiddling and only wound up using their dark blue and NORSE font. I briefly considered doing something with the individual cards, like putting the Odin Logo on them. They can feel a little empty, but it's not worth the struggle with readability. Instead, I put the Odin logo in the background. In my opinion, that looks amazing, so I kept that style for the others.

### Trading Tab

For the Trading Tab, I used a Chart on Tradingview. There's a little easter egg where you can make your chart look like "Tokyo Night". Not always great for trading, but perfect for here. The Borders of the Card are green because... Green is good.

### My Websites Tab

For the 'My Websites' Tab, it's using the Masque de Fer blue. It was going to look black & white, with a black & white photo image that was on the old version of the MDF club website. It turns out, it's _probably_ a photo by Gjon Mili for Life magazine, 1942, depicting Seymour Gross and Arhur Tauber. Unfortunately, I can't find the image online, so it's hard to confirm. Still... It probably means it's bad form to just put it on my website. Too bad; it looked amazing.

## My Websites

I was working on the Masque de Fer Website along the Odin Project. I'm not entirely sure how I feel about it being a plain HTML/CSS/JS endeavour. I could remake it with React, or add a backend with logins, but I'd probably be overdesigning. It's certainly interesting to build with HTML/CSS/JS and then libraries/frameworks like React. The Lich of Leilon campaign website is something that mostly worked out of the box with Obsidian and Quartz. Maybe I'm just showing it because it's cool and the URL is amazing.

## Stories

The setup for these is slightly different, because they don't (all) have a Github Repo to go with them. Most are single scripts, not requiring much to get working, or a separate environment that's a bit outside of the regular wheelhouse. So, instead, I changed the buttons to a single 'Read Me!' button, which'll open up a story in a new SPA Page. If it has a Repo, it's linked there. It has a .MD file and a .PY file which it displays side by side, so the reader can get an idea what I'm talking about. I had considered writing snippets in the text but... It's not worth the effort of turning it into a full blog for each of these.
